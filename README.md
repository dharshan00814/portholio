# CyberSec Portfolio

A React + Vite portfolio that recreates the dark cyber-security themed showcase in this workspace.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- `public/resume-placeholder.txt` is a placeholder for the final resume asset.
- The contact and project actions are wired to local anchors or contact links so the page works without external services.
